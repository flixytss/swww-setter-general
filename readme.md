# Swww-setter-general

I Love SWWW. This library just put a swww background, BUT, you can put plugins when putting a background, The plugins are .lua files, There are two plugins in the directory `plugins`

## Usage

Just do `swww-setter "file"` and the background will be putted and the plugins will be executed.  
The plugin named `save.lua` needs the binary `main-color` that is another repository of my github account: https://github.com/flixytss/MainColor
